SUDOKU INTERATIVO

Este projeto é um trabalho da disciplina de Lógica de Programação I. O programa permite que o usuário jogue um sudoku interativo em que ele pode
posicionar e remover um número de/em uma determinada posição, checar se os números já colocados estão corretos e desfazer as jogadas já realizadas. 
O programa tem a limitação de ter um número fixo de checks possíveis e uma entrada inalterável contendo os jogos possíveis.  

Repositório no GitHub: https://github.com/gabodin/Sudoku-Interativo
